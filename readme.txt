Project Description:
Name: Dungeon Explorer
Description: A dungeon crawler-like game in which a player tries to collect all the hidden treasure scattered across a maze while avoiding/defeating enemies.

How to Run:
Run the file main_game.py in an editor.  Make sure that the audio file "Helltaker OST.mp3" is included in the "code" folder.

Libraries:
Pygame may need to be installed.

Shortcut Commands:
None
